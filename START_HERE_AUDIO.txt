## 🎯 HOW TO GET AUDIO FILES

### Method 1: Generate Local Placeholders ⭐ FASTEST
```
1. Run: bash generate-audio.sh
2. Placeholder .wav and .mp3 files appear in audio/sounds/
3. Use these only for local testing
```

### Method 2: Replace With Licensed Recordings
```
1. Prefer your own recordings, Freesound CC0/CC-BY, or compatible Wikimedia Commons files
2. Keep attribution/license notes for anything you download
3. Replace matching filenames in audio/sounds/
```

### Method 3: Avoid Arbitrary Scraping
```
1. Xeno-Canto is bot-protected and not reliable for unattended scraping
2. Freesound supports licensed downloads, but API access requires login and an API key
3. Random audio sites may not permit reuse or automation
```
